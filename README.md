
# 🚀 AI MLOps Project (AWS + Docker)

## 📌 Overview
This project demonstrates an end-to-end AI system deployed on AWS using DevOps practices.

## 🧠 Features
- Machine Learning prediction model
- REST API using Flask
- Docker containerization
- Deployment on AWS EC2

## ⚙️ Tech Stack
- Python (Flask, Scikit-learn)
- Docker
- AWS EC2
- Linux

## ▶️ Run Project
```bash
docker build -t ai-app .
docker run -p 5000:5000 ai-app
